Instructions

  1. Make a new directory (mkdir) on your machine and change into it (cd)
  2. Select the year and dates of your desired git commits on the 50 x 7 grid above
  3. Copy the auto-generated bash script to your terminal
  4. Execute the script - it will initialize a git repo, readme.md, and make git commits on all your selected dates for that year
  5. Create a new git repository, public or private, on your Github or Gitlab account
  6. Push the local repository with your script-generated commits to your newly created Github or Gitlab repository
  7. Celebrate your inner artist

About

A simple GUI that generates a bash script to make git commits on user-selected dates.

Use it to write/draw/fill in your Github or Gitlab profile commit boxes.
